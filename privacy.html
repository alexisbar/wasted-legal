Politique de confidentialité

1. Collecte des données

Nous ne collectons que les données strictement nécessaires au fonctionnement de l'application :

  • Informations de transaction pour les abonnements Premium
  • Préférences de l'application

2. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • Gérer votre abonnement Premium
  • Améliorer votre expérience utilisateur

3. Stockage des données

Toutes les données sont stockées localement sur votre appareil. Nous n'utilisons pas de serveurs externes.

4. Vos droits

Vous pouvez à tout moment :

  • Accéder à vos données
  • Supprimer l'application et toutes les données associées
